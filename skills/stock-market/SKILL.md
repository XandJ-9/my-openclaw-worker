# Stock Market - 股市行情数据获取

获取A股市场行情数据，支持指数、个股实时行情查询和复盘报告生成。

## 激活条件

当用户提到以下关键词时激活：
- 股市、A股、股票
- 行情、涨跌、复盘
- 指数、个股
- 交易、收盘、开盘

## 功能特性

### 1. 查询指数行情

获取主要A股指数的实时行情数据：
- 上证指数 (sh000001)
- 深证成指 (sz399001)
- 创业板指 (sz399006)
- 中小板指 (sz399005)

### 2. 查询个股行情

查询任意A股个股的实时行情：
- 支持完整代码（如：600718, 000001）
- 支持前缀代码（如：600718, 000001）
- 自动识别上海/深圳交易所

### 3. 生成复盘报告

自动生成A股复盘报告：
- 主要指数表现
- 市场情绪统计
- 基础操作建议

## 使用方法

### 查询指数行情

```
查询今日A股指数行情
获取上证指数、深证成指、创业板指的行情
```

### 查询个股行情

```
查询贵州茅台(600519)的行情
600718现在多少钱
```

### 生成复盘报告

```
生成昨日A股复盘报告
帮我做个市场复盘
```

## 技术实现

- **数据源：** 腾讯财经API (http://qt.gtimg.cn/q=)
- **解析方式：** 正则表达式解析返回的文本格式
- **输出格式：** Markdown + 纯文本

## 依赖管理

使用 uv 管理依赖：
```bash
uv add requests
```

## 注意事项

1. **数据延迟：** 数据可能有5-15秒延迟
2. **交易时间：** 仅在交易时间（9:30-15:00）有实时数据
3. **周末/节假日：** 可能无数据或显示昨日数据
4. **API限制：** 腾讯API对请求频率有限制
